# Baseball simulator

### Build project
1. Install node, yarn
1. Open CLI within project directory
1. In CLI: `yarn install`

### Run local build
1. In CLI: `yarn start`
1. In browser: `localhost:8080`

### Run unit tests
1. In CLI: `yarn test`
